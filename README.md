# Aprendizado Docker: Aplicação Conteinerizada

Este documento registra a conclusão dos meus estudos sobre **Docker** e a ímplementação bem-sucedida de uma  aplicação utilizando containers.

## O que foi aprendido?

Durante este processo, dominei os seguintes conceitos fundamentais:

-[x]  **Docker Engine:** Entendimento da arquitetura e  funcionamento.
-[x]  **Imagens:** Criação e otimização de imagens personalizadas.
-[x]  **dockerfile:** Automação de build da aplicação.
-[x]  **Containers:** Gerênciamento de ciclo de vida (run, stop, rm).
-[x]  **Docker Compose:** - Orquestração de múltiplos serviços (Aplicação + Banco de Dados).
-[x]  **Volumes & Networks:** Persistência de dados e comunicação entre serviços.

## A Aplicação

A aplicação foi totalmente conteinerizada, garantindo que o ambiente de execução seja idêntico em qualquer máquina.

### Como rodar o projeto:

bash
# Clone o repositório
git clone https://github.com

# Entre na pasta
cd seu-projeto

# Suba a aplicação com Docker Compose
